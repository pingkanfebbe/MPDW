Penanganan Autokorelasi Metode Hildreth-Lu

library(HoRM)
## Registered S3 method overwritten by 'quantmod':
##   method            from
##   as.zoo.data.frame zoo
# Hildreth-Lu
hildreth.lu.func<- function(r, model.rlb){
  X <- model.matrix(model.rlb)[,-1]
  y1 <- model.response(model.frame(model.rlb))
  n <- length(y1)
  t <- 2:n
  y1 <- y1[t]-r*y1[t-1]
  X <- X[t]-r*X[t-1]
  
  
  return(lm(y1~X))
}

Simpulan

Berdasarkan hasil analisis di atas, dapat disimpulkan bahwa pada data deret waktu New Delhi AQI terdapat autokorelasi. Hal ini dapat dideteksi setelah melakukan pengujian dengan secara eksploratif melalui metode grafik residual dan ACF PACF serta uji formal Durbin Watson. Oleh karena itu, dilakukan penanganan autokorelasi terhadap data AQI dengan metode Cochrane-Orcutt dan Hildreth-Lu. Setelah dilakukan penanganan menggunakan kedua metode tersebut, masalah autokorelasi pada data New Delhi AQI hanya dapat berhasil diatasi menggunakan metode Cochrane-Orcutt. Sedangkan dengan metode Hildreth-Lu tidak dapat teratasi atau masih terindikasi adanya autokorelasi.
